## 📊 결과

KoBERT 모델은 다음과 같은 하이퍼파라미터를 사용하여 학습을 진행하였습니다.

| 항목 | 값 |
|------|----|
| Learning Rate | 5e-5 |
| Epoch | 5 |
| Batch Size | 200 |

### 학습 결과

| Epoch | Train Accuracy | Test Accuracy |
|------:|---------------:|--------------:|
| 1 | 0.67 | 0.80 |
| 2 | 0.82 | 0.81 |
| 3 | 0.86 | 0.81 |
| 4 | 0.90 | **0.82** |
| 5 | 0.92 | 0.81 |

### 결과 분석

- 학습이 진행될수록 Train Accuracy는 0.67에서 0.92까지 지속적으로 향상되었습니다.
- Test Accuracy는 최대 0.82를 기록하였으며, 이후에는 추가적인 성능 향상이 나타나지 않았습니다.
- Epoch가 증가할수록 학습 데이터에 대한 성능은 향상되었지만 테스트 데이터의 성능은 정체되는 경향을 보여 과적합이 시작되는 양상을 확인할 수 있었습니다.
- 실험 결과를 통해 Epoch 4에서 가장 높은 테스트 성능을 확인하였으며, 모델 학습 시 적절한 학습 횟수(Epoch)를 설정하는 것이 모델의 일반화 성능에 중요한 요소임을 확인할 수 있었습니다.

## ✨ 장문 입력 추론

학습에 사용한 AIHub 데이터셋의 **평균 발화 길이가 약 77자**였기 때문에, 실제 서비스에서 이보다 긴 일기가 입력될 경우에도 학습 데이터와 유사한 입력 형태를 유지할 수 있도록 **77자 단위로 텍스트를 분할하여 추론하는 로직**을 구현하였습니다.

분할된 각 청크는 독립적으로 감정을 예측하며, 이후 각 청크의 감정 확률을 **길이 기반 가중평균(Length-weighted Average)** 방식으로 통합하여 최종 감정 분포를 산출하였습니다.

### 추론 과정

```text
긴 일기 입력
      │
      ▼
77자 단위로 분할
      │
      ▼
각 청크별 KoBERT 감정 예측
      │
      ▼
Softmax를 통한 감정 확률 계산
      │
      ▼
길이 기반 가중평균
      │
      ▼
최종 감정 분포 출력
```

### 추론 예시

#### 입력

> 오늘은 정말 감정의 변화가 컸다. 오전에는 발표 준비를 열심히 했는데 예상보다 좋은 평가를 받아 뿌듯했다. 하지만 오후에는 실수 하나 때문에 팀원들에게 미안한 마음이 들었고 잠시 자신감도 떨어졌다. 퇴근 후 친구와 맛있는 저녁을 먹으며 이야기를 나누다 보니 기분이 다시 좋아졌고 오늘 하루를 잘 마무리할 수 있었다.

#### 출력

| 감정 | 확률 |
|------|------:|
| 슬픔 | **84.17%** |
| 행복 | 12.87% |
| 분노 | 2.19% |
| 공포 | 0.61% |
| 놀람 | 0.06% |
| 중립 | 0.00% |
| 혐오 | 0.00% |

77자를 초과하는 입력도 여러 청크로 분할하여 감정을 예측한 뒤, 길이 기반 가중평균을 적용하여 하나의 감정 분포로 통합하였습니다. 이를 통해 장문의 일기에 대해서도 학습 데이터와 유사한 입력 형태를 유지하면서 감정 분석을 수행할 수 있었습니다.

## 🔧 개선점

- 학습 데이터의 평균 발화 길이에 맞춰 77자 단위로 텍스트를 분할하고 길이 기반 가중평균을 적용하여 장문의 일기를 처리하도록 구현하였습니다.
- 그러나 각 청크를 독립적으로 예측한 후 평균을 계산하는 방식이기 때문에 문맥의 흐름이나 시간에 따른 감정 변화를 충분히 반영하지 못하는 한계를 확인하였습니다.
- 실제 추론 결과에서도 **'뿌듯함 → 미안함 → 행복'**으로 감정이 변화하는 일기를 입력하였음에도 슬픔이 **84.17%**로 가장 높게 예측되는 현상이 나타났습니다. 이는 감정의 흐름보다 각 청크에서 예측된 감정 확률의 평균에 크게 영향을 받기 때문입니다.
- 향후에는 문장 단위 분할 후 Attention 기반 가중치 적용, Hierarchical Transformer 구조, Longformer 등 장문 처리에 특화된 모델을 적용하여 문맥 정보를 유지하고 감정 변화를 보다 효과적으로 반영하는 방향으로 개선할 계획입니다.
- 또한 KLUE-BERT, KoELECTRA 등 다른 한국어 사전학습 모델과 성능을 비교하여 모델의 성능을 검증하고, 데이터셋 확장 및 클래스 불균형 개선을 통해 일반화 성능을 향상시키고자 합니다.
